This project is a Gym Management System backend built with Node.js and Express, designed to streamline the administration of clients, memberships, payments, and internal operations.

The application uses MySQL as the relational database management system, integrated through the mysql2 package to ensure efficient query execution, secure connection handling, and scalable database operations.

The development environment incorporates Nodemon, enabling automatic server restarts during development to improve productivity and workflow efficiency.
